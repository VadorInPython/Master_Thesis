# Multimodal Data Fusion for Vessel Tracking

This project focuses on multimodal data fusion from sources such as video frames, AIS and acoustic signals received from the underwater cable. 

The **main goal** is to develop software that allows users to **track vessels continuously** based on the aforementioned data, even when some of these sources are interrupted or fully cut-off. 

Throughout this project, I plan to use **Python**, **PyTorch** and other relevant libraries.

---

### Project Roadmap
*The steps of this project are outlined below and will be continuously updated after successful iterations.*

#### **Phase 1: Modality Alignment**
Mapping independent sensor data to a **common geographical reference framework** (latitude/longitude coordinates) to enable **spatio-temporal correlation** between the modalities.
